QAbstractAudioOutput¶

Detailed Description¶

The PySide.QtMultimedia.QAbstractAudioOutput class provides access for PySide.QtMultimedia.QAudioOutput to access the audio device provided by the plugin.

PySide.QtMultimedia.QAbstractAudioOutput implements audio functionality for PySide.QtMultimedia.QAudioOutput , i.e., PySide.QtMultimedia.QAudioOutput routes function calls to PySide.QtMultimedia.QAbstractAudioOutput . For a description of the functionality that is implemented, see the PySide.QtMultimedia.QAudioOutput class and function descriptions.

PySide.QtMultimedia.QAbstractAudioOutput.start(device)¶
Parameters:device – PySide.QtCore.QIODevice
Return type:PySide.QtCore.QIODevice

Uses the device as the PySide.QtCore.QIODevice to transfer data. If device is null then the class creates an internal PySide.QtCore.QIODevice . Returns a pointer to the PySide.QtCore.QIODevice being used to handle the data transfer. This PySide.QtCore.QIODevice can be used to write() audio data directly. Passing a PySide.QtCore.QIODevice allows the data to be transferred without any extra code.
